# Autoencoder for Image Restoration

## Project Overview

This project implements a convolutional autoencoder for image restoration using deep learning.

The model learns to reconstruct clean images from corrupted (noisy or blurred) inputs using a CNN-based encoder–decoder architecture with skip connections.

## Features

- Convolutional Autoencoder
- Image Denoising
- Image Deblurring
- Skip connections architecture
- PSNR and SSIM evaluation
- Early stopping training
- Visualization of results
  
## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Fashion-MNIST dataset

## Dataset

Fashion-MNIST dataset from TensorFlow.
